# BKAmos.github.io

Static Jekyll site for SciAna Consultancy.

## Cloudflare Pages deployment

This repo is configured to build with Jekyll on Cloudflare Pages.

Use these settings when creating the Pages project:

- Framework preset: `None`
- Build command: `bundle exec jekyll build`
- Build output directory: `_site`
- Root directory: `/` (repo root)

For detailed instructions (including custom domain setup), see:

- [`CLOUDFLARE_PAGES_SETUP.md`](./CLOUDFLARE_PAGES_SETUP.md)
